| cheatnotes =
*{{Icon|Aura}} ''"Ye must sink the sams to learn me code"'' - Drowned Dan will cast an ''Invulnerability'' -300% Brace Aura making him Immune to Damage.
*{{Icon|Round}} ''"The code is #-#-#-#"'' - At the start of the second Round, Drowned Dan will reveal the first number of the code. Each time a Minion is defeated, the next number of the code is revealed at the start of the following Round. The code represents four different Spell pip costs that Drowned Dan must be targeted with in order to deactivate the Invulnerability Aura.
'''List of Codes:'''
*Drowned Dan will use one of the following nine codes:
**"1-5-2-7"
**"1-5-7-6"
**"1-5-7-7"
**"3-4-0-6"
**"3-4-7-5"
**"3-4-7-6"
**"4-7-0-1"
**"4-7-3-2"
**"4-7-3-6"
'''Correct Responses:'''
*''"Blast! Ye be correct!"'' - is his response if answered correctly the first time.
*''"Yar! A lucky guess!"'' - is his response if answered correctly the second time.
*''"Blimey! They've figured it out!"'' - is his response if answered correctly the third time.
*''"Nooo! My protection!"''' - is his response if answered correctly the fourth time, and casts a version of Supernova on himself that deals no Damage.
'''Incorrect Responses:'''
*''"Incorrect! Back to the start with ye!"'' - is his response if answered incorrectly, starting over the original sequence.
