| cheatnotes =
'''Tiki'''
*{{Icon|Late}} ''"You should be ashamed!"'' - If a Wizard joins the duel after it has already begun, Tiki will cast {{Link|Spell|Guiding Light}} on his teammate(s) and an {{Link|Spell|Aegis}} version of {{Link|Spell|Spirit Armor}} on a random ally.
*{{Icon|Damage over Time}} ''"Let's spice things up!"'' - If a DoT is cast on anyone in the duel, he will cast {{Link|Spell|Lifeblade}}, remove the DoT, and heal with Sprite Swarm (2 Pip).
:*Pet spells will trigger this cheat.

'''Ricky'''
*{{Icon|Late}} ''"Cheater!"'' - If a Wizard joins the duel after it has already begun, then Ricky will triple-cast {{Link|Spell|Healing Current}} on himself, healing 100, 400, then 1,000 Health.
*{{Icon|Blade}} ''"Nice try!"'' - If a Wizard casts a [[:Category:Blade Spells|Blade]] spell, then Ricky will cast {{Link|Spell|Steal Charm}} on the Wizard that received the Blade.
:*{{Link|Spell|Quench}} can be used to dispel this cheat.
:*Aegis can be used to prevent removal.
:*If a tri-Blade spell is used, then only the last part of the Blade will be removed (Fire for {{Link|Spell|Elemental Blade}}s and Life for {{Link|Spell|Spirit Blade}}s).
:*If a multi-target blade spell is used (e.g. {{Link|Spell|Bladestorm}}), only the Wizard who cast the Spell will be affected.
:*Steal Charm can be used to steal back the Blade without a penalty.
:*{{Link|Spell|Supercharge}} does not trigger this cheat.
